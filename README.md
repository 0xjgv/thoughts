# Portfolio Thoughts Starter

- MDX and Markdown support
- Optimized for SEO (sitemap, robots, JSON-LD schema)
- RSS Feed
- Dynamic OG images
- Syntax highlighting
- Tailwind v4
- Vercel Speed Insights / Web Analytics
- Geist font

## How to Use

Develop the app:
```bash
pnpm dev
```

Deploy the app:
```bash
pnpm deploy
```
